<h3>Ideia principal</h3>
O projeto consiste em uma tela de login mobile simples e moderna, focada no acesso de alunos através do "RM" e "Senha". A interface utiliza um tema escuro com detalhes em rosa (identidade visual da FIAP) e oferece uma experiência interativa, com feedbacks visuais como a simulação de carregamento (loading) ao fazer login.

<h3>Requisitos</h3>
Para a construção desta tela, o projeto cumpriu os seguintes requisitos:

* **Funcionalidades:**
  * Campos de entrada para "RM" (teclado numérico) e "Senha".
  * Botão prático para limpar o campo de RM quando houver texto.
  * Alternador para "Mostrar/Ocultar" a senha digitada.
  * Opção de "Lembrar-me" controlada por um Switch.
  * Simulação de carregamento (Spinner) ao clicar no botão de Entrar.

* **Estrutura e Visual:**
  * Desenvolvimento feito em **React Native**.
  * Estilização completa utilizando **NativeWind** (Tailwind CSS).
  * Utilização de todos os componentes base exigidos:
    * `Text` (Textos e rótulos), `TextInput` (Campos de formulário), `TouchableOpacity` (Botões), `Image` (Logo) e `Switch` (Lembrar-me).
